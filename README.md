## 📖 概要

MemoBlendは日記が書けるWebアプリケーションです。
テキストはもちろん、画像や映像、音声ファイルなどで日々を記録することができます。
また、記録した内容を解析して、あなたがどのような生活をしたのかを振り返ることができます。
